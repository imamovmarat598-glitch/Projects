## Фирменный стиль

```css

/* Градиенты */

/* Нейтральные */
Background:  #FFFFFF

```

Основной:    SF Pro Text / Roboto Regular

## App Icon Design

### Концепция 1: "GPS Pin + Earth"

```
┌─────────────────────┐
│    [  🌍  ]         │  Синий круг (земной шар)
│      \/             │  GPS-пин поверх
│    📍              │  Градиент синий → зеленый
│                     │
│   GeoMark           │
└─────────────────────┘
```

**Описание:**
- Фон: градиент от синего (#4A90E2) к бирюзовому (#00f2fe)
- Центр: стилизованная планета Земля (полупрозрачная)
- Поверх: красный GPS-пин (#FF6B6B)
- Стиль: плоский дизайн, минимализм

### Концепция 2: "Satellite + Camera"

```
┌─────────────────────┐
│    🛰️               │  Спутник в углу
│       📷            │  Камера в центре
│     ───────         │  Координатная сетка
│                     │
└─────────────────────┘
```

**Описание:**
- Фон: темно-синий градиент
- Координатная сетка (latitude/longitude lines)
- Иконка камеры с GPS-чипом
- Спутник Роскосмоса в углу

### Концепция 3: "Map Marker" (Рекомендуемая)

```
┌─────────────────────┐
│                     │
│      ╱█╲            │  Стилизованный маркер
│     ╱ • ╲           │  С точкой GPS внутри
│    ╱     ╲          │  Градиентная заливка
│   └───────┘         │
│                     │
└─────────────────────┘
```

**Описание:**
- Форма: классический map pin
- Цвет: градиент от красного (#FF6B6B) к оранжевому (#FFA500)
- Внутри: белая GPS-точка с пульсацией
- Фон: светлый радиальный градиент
- Тень: мягкая, для объема

## Файлы для создания

### 1. App Icon (1024x1024px)

**Требования:**
- Формат: PNG (без прозрачности)
- Размер: 1024x1024px
- Цветовое пространство: sRGB
- Без скругления углов (добавляется автоматически)
- Без текста внутри иконки

**Где использовать:**
- App Store Connect
- Google Play Console
- RuStore Console

### 2. Adaptive Icon (Android)

**Foreground (432x432px):**
- Основной элемент иконки
- Центрирован в безопасной зоне (264x264px)

- Сплошной цвет или градиент
- Будет обрезаться по маске устройства

## Инструкция по созданию иконки

### Вариант 1: Figma (рекомендуется)

1. Откройте Figma
2. Создайте Frame 1024x1024px
3. Примените градиент:
   - Тип: Radial
   - Цвета: #4A90E2 → #00f2fe
4. Добавьте GPS-пин:
   - Используйте плагин "Ionicons"
   - Цвет: #FF6B6B
   - Размер: 512px
   - Позиция: центр
5. Добавьте тень:
   - Blur: 20px
   - Y: 10px
   - Opacity: 30%
6. Экспорт: PNG, 1024x1024px, @1x

### Вариант 2: Canva (простой)

1. Перейдите на canva.com
2. Создайте дизайн 1024x1024px
3. Используйте шаблон "App Icon"
4. Добавьте элементы:
   - Градиентный круг
   - Иконку локации
   - Настройте цвета
5. Скачайте как PNG

### Вариант 3: AI генерация (DALL-E, Midjourney)

**Prompt для генерации:**
```
App icon for GPS camera application, modern flat design,
blue gradient background, red GPS pin marker in center,
minimalist style, professional, high quality, 1024x1024px,
no text, suitable for iOS and Android
```

## Рекомендации по дизайну UI

### Экран камеры

- Минимум элементов, максимум пространства для превью
- GPS-индикатор всегда виден
- Цветовая индикация точности (зеленый/желтый/красный)
- Большая кнопка съемки (80x80px)

### Карта

- Кластеризация маркеров при zoom out
- Превью фото при клике на маркер
- Переключатель слоев (OSM/Google/Роскосмос)
- Кнопка "Моя локация"

### Галерея

- Grid layout 3 колонки
- Превью 120x120px
- Координаты под каждым фото
- Фильтры: по дате, месту, точности GPS

### Настройки

- Группировка по категориям
- Toggle switches для опций
- Четкие описания каждой настройки
- Premium features выделены цветом
