## 🎨 Стиль и дизайн

### Общая концепция

**Формула успешного скриншота:**
```
Скриншот = Реальный UI (70%) + Маркетинговый текст (20%) + Декоративные элементы (10%)
```

**Цветовая схема:**
- Фон: Градиент (#4A90E2 → #FFFFFF)
- Акцент: #FF6B6B (для важных элементов)
- Текст: #212529 (темный)
- Подсветка: #FFD700 (золотой для "NEW" badges)

## 🛠️ Инструменты для создания

### Вариант 1: Figma (Профессиональный)

**Шаблоны:**
- [Mockuuups Studio](https://mockuuups.studio/)
- [Facebook Design Resources](https://facebook.design/devices)
- [Figma Community](https://www.figma.com/community) - "App Store Screenshots"

**Процесс:**
1. Открыть Figma
2. Импортировать Phone Mockup
3. Вставить скриншоты UI внутрь mockup
4. Добавить текст поверх
5. Применить эффекты (тени, blur)
6. Экспорт: PNG, 2x scale

### Вариант 2: Canva (Простой)

**Шаблоны:**
- Поиск: "App Store Screenshot Template"
- Размер: Custom (1242x2688)
- Добавить Phone mockup из Elements
- Вставить скриншот внутрь
- Добавить текст
- Скачать: PNG, High Quality

### Вариант 3: Shotbot.io (Автоматический)

```
1. Перейти на shotbot.io
2. Загрузить скриншоты UI
3. Выбрать Phone mockup (iPhone 15 Pro Max)
4. Добавить текст с помощью встроенного редактора
5. Выбрать фон (градиент)
6. Скачать готовые mockups
```

### Вариант 4: Placeit.net (Платный)

```
1. Перейти на placeit.net
2. Выбрать "App Screenshots"
3. Выбрать шаблон с текстом
4. Загрузить свои скриншоты
5. Кастомизировать текст и цвета
6. Скачать: $8-12 за комплект
```

## 📝 Получение скриншотов UI

### iOS Simulator

```bash
# Запустить Expo
cd GeoMarkApp
npx expo start --ios

# В симуляторе:
# Cmd + S для скриншота
# Скриншоты сохраняются на Desktop

# Или через терминал:
xcrun simctl io booted screenshot screenshot.png
```

### Android Emulator

```bash
# Запустить
npx expo start --android

# В эмуляторе:
# Ctrl + S (Windows) или Cmd + S (Mac)
# Скриншоты в: ~/Library/Android/screenshots/

# Или через adb:
adb shell screencap /sdcard/screenshot.png
adb pull /sdcard/screenshot.png
```

### Физическое устройство

**iOS:**
1. Подключить iPhone
2. Запустить приложение
3. Volume Up + Power Button
4. AirDrop на Mac

**Android:**
1. Подключить устройство
2. Запустить приложение
3. Volume Down + Power Button
4. Передать через USB

## 🎨 Текстовые overlay (PSD Template)

### Photoshop Actions

Создать Action для быстрого добавления текста:

```
1. Открыть скриншот
2. Создать новый слой "Text Overlay"
3. Добавить градиент фон (linear, top to bottom, white to transparent)
4. Добавить текст (см. спецификацию выше)
5. Добавить тень: Drop Shadow, 50% opacity, 20px blur
6. Сохранить Action
7. Применить ко всем скриншотам: Batch Processing
```

## 📐 Сетка и Safe Areas

### iPhone Safe Areas

```
┌─────────────────┐
│ ↕ 100px         │ ← Отступ от Dynamic Island
├─────────────────┤
│                 │
│  SAFE AREA      │ ← Основной контент
│                 │
├─────────────────┤
│ ↕ 100px         │ ← Отступ от Home Indicator
└─────────────────┘
```

**Не размещайте важный текст:**
- В верхних 100px (Dynamic Island)
- В нижних 100px (Home Indicator)
- По краям (<50px от boundaries)

## ✅ Checklist перед экспортом

### Технический
- [ ] Правильные размеры (1242x2688 для iPhone, 1080x1920 для Android)
- [ ] Формат: PNG (не JPG!)
- [ ] Цветовой профиль: sRGB
- [ ] Размер файла: <5MB каждый
- [ ] Нет артефактов сжатия
- [ ] Четкий текст (не размыт)

### Контентный
- [ ] UI актуален (соответствует текущей версии)
- [ ] Текст читаем на маленьком экране
- [ ] Нет placeholder-контента ("Lorem ipsum")
- [ ] Реальные данные (не "test@test.com")
- [ ] Правильная локализация (русский язык)
- [ ] Нет ошибок в интерфейсе

### Маркетинговый
- [ ] Заголовок привлекает внимание
- [ ] Ключевая фича подсвечена
- [ ] УТП четко видно (Роскосмос!)
- [ ] Call-to-action понятен
- [ ] Скриншоты рассказывают историю (в порядке)

## 🚀 Загрузка в сторы

### App Store Connect

```
1. Login → My Apps → GeoMark
2. App Store → Screenshots and Preview
3. Выбрать Device: iPhone 6.7" Display
4. Drag & Drop 5 screenshots в порядке
5. Выбрать Device: iPhone 5.5" Display
6. Drag & Drop те же 5 screenshots
7. Save
```

### Google Play Console

```
1. Login → All apps → GeoMark
2. Store Presence → Main store listing
3. Phone screenshots: Upload 3-8 screenshots
4. 7-inch tablet: Optional
5. 10-inch tablet: Optional
6. Save changes
```
