# User Experience Design Document (UXD)
## GPS Camera App

**Версия:** 1.0
**Дата:** 14 января 2026
**Автор:** UX/UI Team

---

## 1. Дизайн-принципы

### 1.1 Основные принципы
1. **Простота превыше всего** — интуитивный интерфейс без обучения
2. **Скорость** — минимум тапов до основных действий
3. **Визуальная ясность** — четкая иерархия информации
4. **Нативность** — следование guidelines платформ
5. **Доступность** — удобство для всех пользователей

### 1.2 Дизайн-язык
- **Android:** Material Design 3
- **iOS:** Human Interface Guidelines
- **Адаптивность:** Поддержка различных размеров экранов и ориентаций

## 5. Подробные экраны

### 5.1 Экран: Камера (Camera Screen)

**Назначение:** Съемка фотографий с отображением GPS данных в реальном времени.

#### Layout:
```
┌───────────────────────────────────────┐
│ [< Back]              [Flash] [Flip]  │ ← Top Bar
├───────────────────────────────────────┤
│                                       │
│                                       │
│                                       │
│         Camera Viewfinder             │
│                                       │
│                                       │
│ ┌───────────────────────────────┐     │
│ │ 📍 GPS: 55.7558° N, 37.6173° E│     │ ← GPS Overlay
│ │ ⬆ Alt: 150m  📶 Accuracy: 8m  │     │
│ │ 🕐 14.01.2026 15:32           │     │
│ └───────────────────────────────┘     │
├───────────────────────────────────────┤
│      [Gallery]  [⚪ Capture]          │ ← Bottom Controls
│                                       │
└───────────────────────────────────────┘
```

#### Компоненты:

**1. Top Bar:**
- Кнопка назад (если не главный экран)
- Переключатель вспышки: Auto / On / Off
- Переключение камеры (фронтальная/основная)

**2. GPS Overlay (полупрозрачная панель сверху):**
- Текущие координаты (широта, долгота)
- Высота над уровнем моря
- Точность GPS с цветовым индикатором
- Текущее время и дата

**3. GPS Accuracy Indicator:**
```
�� Excellent (< 10m)
🟠 Good (10-50m)
🔴 Poor (> 50m)
⚫ No GPS
```

**4. Bottom Controls:**
- Превью последнего фото (ведет в галерею)
- Большая круглая кнопка захвата
- Опциональная кнопка для заметки

#### Взаимодействие:
- **Tap на кнопку захвата** → Сделать фото
- **Long press на кнопку захвата** → Серия фото (burst mode - Phase 2)
- **Tap на превью** → Открыть галерею
- **Pinch gesture** → Zoom камеры

#### States:
- **Loading GPS:** Показать спиннер на GPS панели
- **No GPS:** Предупреждение "GPS недоступен"
- **Photo captured:** Анимация вспышки экрана

│ └─────┴─────┴─────┴─────┘             │

└───────────────────────────────────────┘
```

#### Компоненты:

**1. Header:**
- Заголовок "Мои фото" + счетчик
- Кнопка поиска
- Кнопка фильтров

**2. Quick Filters:**
- Dropdown: Период (Сегодня, Неделя, Месяц, Год, Все)
- Dropdown: Категории (Работа, Путешествия, Семья, Все)
- Кнопка "Сбросить фильтры" (если активны)

**3. Photo Grid:**
- 3 колонки на телефоне, 4-5 на планшете
- Квадратные миниатюры

- **Tap на фото** → Открыть детальный просмотр
- **Long press** → Режим множественного выбора
- **Pull to refresh** → Обновить список
- **Infinite scroll** → Подгрузка старых фото

## 7. User Flows

### 7.1 Основной flow: Съемка и просмотр

```
Запуск приложения
       ↓
[Camera Screen] ← Главный экран
       ↓
Ожидание GPS фикса (1-5 сек)
       ↓
GPS получен → Показать координаты
       ↓
Пользователь нажимает [Capture]
       ↓
Фото сохранено с GPS метаданными
       ↓
Анимация успеха + звук
       ↓
Опции:
  → Продолжить съемку
  → Открыть последнее фото [Tap on preview]
       ↓
  [Photo Detail Screen]
       ↓
  Просмотр метаданных, редактирование заметок
       ↓
  [Показать на карте] → [Map Screen]
```

### 7.2 Flow: Организация фото

```
[Gallery Screen]
       ↓
Long press на фото
       ↓
[Multi-select mode]
       ↓
Выбрать несколько фото
       ↓
Опции:
  → Добавить метки
  → Изменить категорию
  → Экспортировать
  → Удалить
```

### 7.3 Flow: Экспорт данных

```
[Photo Detail Screen] или [Gallery Multi-select]
       ↓
Нажать [Экспорт]
       ↓
[Export Dialog]
  ├─ Выбрать формат: KML / GPX / CSV / ZIP
  ├─ Включить фото: ✓ / ✗
  └─ Включить метаданные: ✓ / ✗
       ↓
Нажать [Экспорт]
       ↓
Генерация файла (показать progress)
       ↓
[Share Sheet] → Выбрать приложение
       ↓
Успешно экспортировано
```

## 8. Адаптивность и ориентация

### 8.1 Portrait Mode (Портретная ориентация)
- Основной режим использования
- Все экраны оптимизированы для портрета
- Camera Screen использует весь экран

### 8.2 Landscape Mode (Альбомная ориентация)
- **Camera Screen:** Поддерживается, UI адаптируется
- **Gallery Screen:** 4-5 колонок вместо 3
- **Map Screen:** Полноэкранная карта
- **Photo Detail:** Фото слева, метаданные справа

### 8.3 Tablet Support
- Увеличенное количество колонок в галерее
- Split-view на некоторых экранах
- Оптимизация для больших экранов

---

## 9. Accessibility (Доступность)

### 9.1 Screen Reader Support
- Все кнопки имеют accessibility labels
- Фото описываются через метаданные
- Навигация поддерживает TalkBack / VoiceOver

### 9.2 Размеры элементов
- Минимальный размер кнопок: 44x44 dp/pt
- Достаточные отступы между элементами
- Крупный текст для важной информации

### 9.3 Контрастность
- Текст на фоне: минимум 4.5:1
- Важные элементы: 7:1
- Поддержка темной темы (Phase 2)

### 9.4 Альтернативные способы ввода
- Голосовой ввод для заметок
- Поддержка клавиатуры на планшетах

---

## 10. Анимации и переходы

### 10.1 Transitions между экранами
- Slide transition для навигации
- Fade для модальных окон
- Длительность: 300ms

### 10.2 Micro-animations
- Ripple effect на кнопках (Android)
- Haptic feedback при действиях
- Loading spinners для долгих операций
- Success animations (✓) после действий

### 10.3 Camera animations
- Вспышка экрана при съемке
- Zoom анимация при pinch
- Smooth focus transitions

## 14. Responsive Design Guidelines

### 14.1 Breakpoints
```
Small Phone:   < 360dp width
