# Подробная инструкция: Запуск GeoMark на iPhone/iPad

## Способ 1: Expo Go (Самый простой - 5 минут)

Это самый быстрый способ запустить приложение на iPhone без Apple Developer аккаунта.

### Шаг 1: Скачайте Expo Go на iPhone

1. Откройте **App Store** на вашем iPhone
2. Нажмите **Поиск** (внизу справа)
3. Введите **"Expo Go"**
4. Найдите приложение от "Expo Project" (фиолетовая иконка)
5. Нажмите **Загрузить** → подтвердите Face ID/Touch ID
6. Дождитесь установки

### Шаг 2: Запустите проект на компьютере

Откройте **PowerShell** или **Terminal** на компьютере:

```powershell
# Перейдите в папку проекта
cd c:\Users\ASuS\Documents\Projects\Projects\GeoMarkApp

# Запустите Expo
npx expo start
```

После запуска вы увидите:
```
Metro waiting on exp://192.168.x.x:8081
› Scan the QR code above with Expo Go (Android) or the Camera app (iOS)

› Web is waiting on http://localhost:8081

› Press a │ open Android
› Press w │ open web
› Press j │ open DevTools
› Press r │ reload app
› Press s │ switch to Expo Go
```

И QR-код в терминале.

### Шаг 3: Отсканируйте QR-код на iPhone

**ВАЖНО: iPhone и компьютер должны быть в одной Wi-Fi сети!**

1. Откройте стандартное приложение **Камера** на iPhone
2. Наведите камеру на QR-код в терминале
3. Появится желтое уведомление сверху: **"Открыть в Expo Go"**
4. Нажмите на это уведомление
5. Откроется Expo Go и начнется загрузка приложения

Готово! Приложение запустится на вашем iPhone.

### Если QR-код не сканируется:

**Вариант А: Введите URL вручную**
1. Откройте Expo Go на iPhone
2. Нажмите **"Enter URL manually"**
3. Введите адрес из терминала, например: `exp://192.168.1.100:8081`
4. Нажмите **"Connect"**

**Вариант Б: Используйте tunnel (если разные сети)**
```powershell
npx expo start --tunnel
```
Это создаст публичный URL, работающий через интернет.

## Способ 2: Веб-версия в браузере (Мгновенно)

Expo позволяет запустить приложение прямо в браузере:

```powershell
cd c:\Users\ASuS\Documents\Projects\Projects\GeoMarkApp
npx expo start --web
```

Или после `npx expo start` нажмите клавишу **w**.

Откроется браузер с адресом `http://localhost:8081`

**Ограничения веб-версии:**
- Камера работает (если браузер поддерживает)
- GPS работает
- Нет доступа к галерее устройства
- Дизайн может отличаться

**Для демонстрации клиентам:**
1. Запустите `npx expo start --tunnel --web`
2. Поделитесь публичным URL с клиентами
3. Они откроют его в браузере на любом устройстве

## Способ 3: Snack (Онлайн без установки)

Expo Snack - это онлайн-редактор, где можно запустить React Native приложение прямо в браузере.

### Шаг 1: Создайте Snack

1. Перейдите на https://snack.expo.dev
2. Вы увидите онлайн-редактор кода
3. Справа есть превью: **"My Device"**, **"Android"**, **"iOS"**, **"Web"**

### Шаг 2: Загрузите код

Скопируйте код из наших файлов в Snack. Или я могу создать готовый Snack с демо-версией.

### Шаг 3: Запустите на iPhone

1. В Snack нажмите **"My Device"**
2. Появится QR-код
3. Отсканируйте его через Expo Go на iPhone

**Преимущество:** Не нужен компьютер с Node.js - всё работает в браузере!

## Способ 4: TestFlight (Для публичного тестирования)

TestFlight - официальный способ Apple для бета-тестирования. Требует Apple Developer аккаунт ($99/год).

### Требования:
- Mac с macOS
- Xcode (бесплатно из App Store)
- Apple Developer Program ($99/год)

### Шаги:

```bash
# 1. Установите EAS CLI
npm install -g eas-cli

# 2. Авторизуйтесь в Expo
eas login

# 3. Настройте Apple credentials
eas credentials

# 4. Соберите для iOS
eas build --platform ios --profile production

# 5. Отправьте в App Store Connect
eas submit --platform ios
```

После этого:
1. Зайдите в App Store Connect
2. Выберите приложение → TestFlight
3. Добавьте тестировщиков по email
4. Они получат приглашение и установят через TestFlight

## Способ 5: Ad-Hoc Distribution (Для ограниченного круга)

Позволяет установить приложение напрямую на конкретные iPhone (до 100 устройств).

### Шаги:

1. Соберите UDID устройств тестировщиков
2. Добавьте их в Apple Developer Portal
3. Создайте Ad-Hoc provisioning profile
4. Соберите IPA:
```bash
eas build --platform ios --profile preview
```
5. Раздайте IPA через Diawi, AppCenter или email

## Частые проблемы на iOS

### "Unable to connect"
```
Причина: iPhone и компьютер в разных Wi-Fi сетях
Решение:
1. Подключите оба к одному Wi-Fi
2. Или используйте: npx expo start --tunnel
```

### "Something went wrong"
```
Причина: Кэш Expo
Решение:
1. В Expo Go: потяните вниз для обновления списка
2. На компьютере: npx expo start -c (очистка кэша)
```

### Камера не работает в Expo Go
```
Причина: iOS ограничения в Expo Go
Решение:
1. Проверьте разрешения в Настройки → Expo Go → Камера
2. Или создайте development build через EAS
```

### Приложение медленно загружается
```
Причина: Большой bundle
Решение:
1. Используйте tunnel: npx expo start --tunnel
2. Или production build через EAS
```
